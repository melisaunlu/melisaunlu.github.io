I have two Web Dev projects I worked on. These include this website that is currently in progress (both front end and backend) and a web application for my database class. Here, I'll show the web application we developed and some screenshots.

For this project, we worked as a team to build a web app. I mainly worked on the database and integrating it with the frontend along with unit tests and coverage reports.

The app runs on React and Vite, with three servers working in the background (Frontend, Backend, and Database). It uses SQLite (volunteer.db) with Flask-SQLAlchemy as the ORM. The database file is auto-created on startup, and models are defined and seeded accordingly.
