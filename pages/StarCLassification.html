Machine Learning Project

Star Classification
Using SDSS Data

Classified stars, galaxies, and quasars using Sloan Digital Sky Survey observational data containing 100,000 samples and 18 features.

Applied preprocessing techniques including label encoding, feature scaling, and feature selection through Random Forest importance ranking.

Built and trained a neural network model achieving approximately 98% classification accuracy.
