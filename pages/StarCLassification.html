Star Classification

This project classifies celestial objects (stars, galaxies, quasars) using SDSS observational data with 100,000 entries and 18 features.

Data preprocessing included label encoding, scaling, and feature selection using Random Forest importance ranking.

A neural network achieved ~98% accuracy in classification.
