Machine Learning Project

Star Classification Using
SDSS Survey Data

Developed machine learning models to classify stars, galaxies, and quasars using Sloan Digital Sky Survey data with over 100,000 astronomical objects.

Applied preprocessing including feature scaling, label encoding, exploratory analysis, and Random Forest feature importance to identify key predictors.

Built and trained a TensorFlow neural network achieving approximately 98% classification accuracy.
