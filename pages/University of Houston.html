Machine Learning & Geophysics

This six-month research project with Dr. Sun focused on machine learning models applied to geophysical resistivity data, with emphasis on California groundwater systems.

The project generates synthetic resistivity profiles using a vertical grid, then trains a neural network through supervised learning.

This work introduced computational geophysics, numerical modeling, and machine learning inversion techniques.
