Machine Learning & Geophysics

This six-month research project with Dr. Sun focused on machine learning applied to geophysical resistivity data, with emphasis on California groundwater systems.

The model generates synthetic resistivity profiles using a vertical grid and trains a neural network through supervised learning.

The work introduces computational geophysics, numerical modeling, and machine learning inversion techniques.
