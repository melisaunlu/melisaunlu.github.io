Interstellar Route Optimization

Full-Stack · 3D Graphics · Real-Time · Pathfinding

A full-stack 3D space travel management system — Google Maps meets an airline booking system for interplanetary travel. Built with Node.js, PostgreSQL, React, Three.js, and Socket.IO.

  • Dijkstra's algorithm computing optimal routes across three cost functions: fastest, safest, and cheapest (fuel calculation)
  • JWT authentication with three role levels (Admin, Pilot, Viewer) and protected REST API endpoints
  • Real-time mission tracking via Socket.IO with live ship position updates and status alerts
  • Interactive 3D galaxy map with clickable solar system planets, smooth camera pathfinding, and planet surface exploration
  • Procedurally generated planet surfaces — unique terrain, atmosphere, flora and fauna per planet type using Three.js
  • Flyable player ship with WASD controls, exhaust trail particle system, and planet landing mechanics
