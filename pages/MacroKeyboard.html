Custom Mechanical Macro Keyboard

  • Hand-wired a 12-key macropad using mechanical switches, diodes, and an ATmega32u4 microcontroller, soldering matrix circuitry (rows/columns) for reliable keypress detection
  • Flashed custom QMK firmware via QMK Toolbox using the HalfKay bootloader process, defining keymaps and layout for a dedicated function/navigation macropad
  • Debugged hardware-firmware integration issues, including switch bounce and miswiring, using a multimeter and soldering iron rework

Compact Battery-Powered Keyboard

  • Assembled a custom keyboard PCB integrating an RP2040-class controller, USB-C charging circuit, and LiPo battery management on a multi-board protoboard stack
  • Soldered SMD and through-hole components, including connector headers and short JST-style wiring harnesses linking the controller, power, and daughterboard stack
  • Fit the electronics into a custom anodized aluminum case with a clean screw-hole pattern, balancing component placement with wiring routing
