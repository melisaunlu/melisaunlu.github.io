Machine Learning & Geophysics

This six-month research project with Dr. Sun was my first formal research experience. I worked on applying machine learning models to geophysical resistivity data, with a focus on California groundwater systems.

The project generates synthetic resistivity profiles using a vertical grid, which are then used to train a neural network. The pipeline includes data generation, interpolation, and supervised training for resistivity prediction.

This work introduced me to computational geophysics, numerical modeling, and machine learning-based inversion techniques.
