Star Classification

This project uses machine learning techniques to classify celestial objects, specifically stars, galaxies, and quasars, using observational data from the Sloan Digital Sky Survey (SDSS). The dataset contains 100,000 entries, each with 18 features such as sky coordinates, brightness in different wavelength filters, and redshift.

The goals are to explore and visualize the data to understand feature distributions. The data is preprocessed by encoding labels, scaling features, and removing unnecessary columns. The most important features are identified using Random Forest feature importance ranking. Predictive models are trained using the top features and evaluated based on accuracy and classification metrics.

The results highlight top predictive features such as redshift and brightness in near-infrared and visible bands (i, r, g). The neural network model achieved approximately 98% accuracy in classifying stars, galaxies, and quasars.
